Useful regular expression search arguments.

To search all infusion damage types: ELEMPHYSDAMAGE|DAMAGEDARK|DAMAGEFIRE|DAMAGEMAGIC|DAMAGETHUNDER

To search all elemental scaling types: INTSCALE|FAITHSCALE

---------------------------------------------------------------------------------------------------------

Before using this script, search and replace the following "variables" with your desired parameters 

SORTBASEID :This is your weapon's base sort ID, it is achieved by removing the last 2 digits from the main weapon's sort ID
           :Example, if your sortId is 50400 then your SORTBASEID would be 504

BASEWEPID :This is your weapon's base ID. You will find it by removing 4 digits from the end of your EquipParamWeapon row ID. 
          :Example, if your row ID is 22180000 then your BASEWEPID will be 2218

INTSCALE :This is your intelligence scaling, for common usage INTSCALE and FAITHSCALE should be set to the same value.

FAITHSCALE :This is your intelligence scaling, for common usage INTSCALE and FAITHSCALE should be set to the same value.

RAWDAMAGE :This is the base damage that will applied to fire, physical and dark in raw, fire and deep infusions.

ELEMPHYSDAMAGE :This is your physical damage to be used on elemental infusions, physical infusions except blessed will keep their original damage

DAMAGEDARK :This is your dark damage, for common usage. This should be set the same as ELEMPHYSDAMAGE

DAMAGEFIRE :This is your fire damage, for common usage. This should be set the same as ELEMPHYSDAMAGE

DAMAGEMAGIC :This is your magic damage, for common usage. This should be set the same as ELEMPHYSDAMAGE

DAMAGETHUNDER :This is your lightning damage, for common usage. This should be set the same as your ELEMPHYSDAMAGE
